¿Que es HTML?

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje fundamental para crear y estructurar páginas web. Define la estructura y el contenido de un documento web, utilizando etiquetas para marcar diferentes secciones como párrafos, encabezados, listas, imágenes y enlaces.

Ejemplos de etiquetas:

p :Define un párrafo. h1, h2, h3, h4, h5, h6: Definen encabezados de diferentes niveles. a: Crea un enlace a otra página web o sección de la misma. img: Inserta una imagen en la página. div, span: Se utilizan para agrupar elementos y aplicar estilos. ol, ul, li: Definen listas ordenadas y no ordenadas.
