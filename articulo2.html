¿Que es Css?

CSS (Cascading Style Sheets), o "Hojas de Estilo en Cascada", es un lenguaje utilizado para dar estilo a los elementos de una página web. Permite definir la apariencia visual, como colores, fuentes, tamaños, espacios, etc., separándola del contenido estructural (HTML).

Propiedades principales:

  • Texto
  • Colores
  • Fondos
  • Bordes
  • Posicionamiento
  • Disposicion
